## ğŸš€ Overview

**TejasGPT** is a frontend-only AI chatbot with a distinct Hinglish + Marathi personality.  
It provides short, friendly, casual replies while retaining context using a chat memory system.  
The chatbot communicates with Google Gemini's API and renders beautiful chat bubbles on a premium UI.

It is specificaly created to help students in following subjects:
- Computer Organization and Architecture  
- Mathematics 
- Digital Electronics and Logic design  
- Software Project Management  
- Object Oriented Programming
